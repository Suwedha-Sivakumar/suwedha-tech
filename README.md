# Personal Portfolio Website – Suwedha Sivakumar

A premium, modern, fully responsive personal portfolio website showcasing professional skills, educational milestones, and work history. Built using a dark theme aesthetic with glassmorphic styles, custom background particles, and smooth scroll animations.

## 🚀 Features

- **Premium UI/UX:** Clean dark mode theme, glassmorphic layout panels, glowing ambient gradients, and smooth hover effects.
- **Dynamic Preloader:** Interactive custom text spinner showcasing the system loading progress.
- **Interactive Tech Arsenal:** Detailed categorical layout for programming, web development, cybersecurity, and administrative multitasking capabilities.
- **Milestones Timeline:** Styled vertical timeline path featuring professional journey updates.
- **Placeholder Layouts:** Ready-to-go clean structures for future Projects and Certifications additions.
- **Interactive Contact Map:** Live embedded Google Map displaying the custom region (Piliyandala, Sri Lanka).

## 🛠️ Built With

- **HTML5 & CSS3** - Custom structured markup and responsive styles.
- **Bootstrap 5** - Responsive layout grid and components.
- **Font Awesome 6** - Clean icon integrations.
- **Particles.js** - Interactive background node networks.
- **Typed.js** - Dynamic animated typing effect in the hero header.
- **GSAP & AOS** - High-performance scroll-triggered visual animations.

## 💻 Local Preview

To run the portfolio site locally on your machine:

1. Open your terminal in the project directory.
2. Spin up a lightweight local server (using Python):
   ```bash
   python -m http.server 8080
   ```
3. Open your browser and navigate to:
   ```
   http://localhost:8080
   ```
